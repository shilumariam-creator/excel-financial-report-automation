#  Excel Financial Report Automation (VBA)

##  Overview
This project is an Excel-based financial automation tool developed using VBA. It processes raw transaction data and automatically generates structured financial reports and visual dashboards.
The tool simulates real-world financial workflows such as data import, cleaning, aggregation, and reporting — similar to tasks performed in business analytics and research environments.

##  Features
*  Automated CSV data import
*  Data cleaning (duplicate removal, formatting)
*  Expense analysis by category
*  Monthly expense trend detection (time-series analysis)
*  Automatic identification of highest spending month
*  Dashboard with charts (category distribution, trends)
*  One-click execution via VBA macro

##  Technologies Used

* Microsoft Excel
* VBA (Visual Basic for Applications)
* Pivot Tables
* Basic data processing techniques

##  How to Use

1. Download or clone this repository
2. Open `financial\_tool.xlsm` in Excel
3. Enable macros when prompted
4. Click the \*\*"Run Financial Report"\*\* button
5. View results in:
     * Summary sheet
     * Dashboard sheet
